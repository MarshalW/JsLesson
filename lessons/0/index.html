JavaScript
应用开发

课程目标

  • 运用JavaScript独立编写基本的Web App

    JavaScript易于编写，从客户端到服务器端都使用它，可大幅提高开发效率

  • 理解复杂交互的架构模型，为大型应用开发做技术准备

    客户端的交互越来越复杂，需要有正确的架构使之易于维护

  • 掌握部分复杂交互的开发和设计技巧

    通过具体的翻页和缩放组件，说明开发和设计技巧

课程内容

  • JavaScript语言语法

    在需要的地方穿插语言和语法介绍，避免枯燥的大段讲解

  • 客户端的JavaScript

    如何在客户端实现可复用的复杂交互以及平滑的动画等

  • 服务器端的JavaScript

    如何编写服务器端代码，并用NoSQL的方式持久化对象

对听众的要求

  • 了解html, css和javascript

    能编写简单的html，css和javascript代码

  • 使用过jQuery

    这不是必须的，但对理解我们的示例有好处

需要提前准备的事情

  • Chrome或Safari浏览器

    因为移动应用开发中主要针对WebKit内核浏览器

  • WebStorm

    IDE，对JavaScript（包括服务器端的Node.js）支持的很好

  • Mac环境下安装Xcode

    客户端动画部分需要使用到iPad模拟器

  • 安装git

    课程示例均在GitHub上，需要通过git获取

如何使用示例

  • 使用git下载源代码

    git clone https://MarshalW@github.com/MarshalW/HelloExpress.git

  • 切换示例的版本

    git checkout v0.2

  • 项目在WebStorm中打开

课程分阶段安排

  • JavaScript应用开发入门

    讲解客户端、服务器端和数据库技术使用的最小集，快速入门。
    课时安排：6小时

  • 客户端复杂交互实践

    讲解复杂交互架构和复用方式和交互的具体实现方式
    课时安排：6小时

  • 服务器端的实践

    讲解JavaScript在服务器端的最佳实践，包括数据库部分
    课时安排：6小时

每个阶段可选一个同等学时数的作业实践指导课程，由专人支持辅导。
